Build script

The OTT reference taxonomy is an algorithmic combination of several source taxonomies. For code, see the source code repository.

Sources

The following information about sources is up to date as of 14 January 2014. This page is updated manually and what you see here may not immediately reflect the status of the very latest taxonomy version.

  1. Taxonomy from: DS Hibbett, M Binder, JF Bischoff, M Blackwell, et al. A higher-level phylogenetic classification of the Fungi. Mycological Research 111(5):509-547, 2007. With updates through 2014, archived at http://dx.doi.org/10.6084/m9.figshare.915439.
  2. Taxonomy from SILVA 16S ribosomal RNA database, version 115. Web site: http://www.arb-silva.de/download/arb-files/ See: Quast C, Pruesse E, Yilmaz P, Gerken J, Schweer T, Yarza P, Peplies J, Glöckner FO (2013) The SILVA ribosomal RNA gene database project: improved data processing and web-based tools. Nucleic Acids Research 41 (D1): D590-D596.
  3. Index Fungorum, from database query result files provided by Paul Kirk, April 2014 (personal communication). Web site: http://www.indexfungorum.org/
  4. Taxonomy from: Schäferhoff, B., Fleischmann, A., Fischer, E., Albach, D. C., Borsch, T., Heubl, G., and Müller, K. F. (2010). Towards resolving Lamiales relationships: insights from rapidly evolving chloroplast sequences. BMC evolutionary biology 10(1), 352.
  5. NCBI Taxonomy, from the US National Center on Biotechnology Information. Updated frequently; for OTT 2.8 we used a version dated 11 June 2014. Web site:http://www.ncbi.nlm.nih.gov/Taxonomy/
  6. GBIF Backbone Taxonomy, from the Global Biodiversity Information facility. Location: http://www.gbif.org/dataset/d7dddbf4-2cf0-4f39-9b2a-bb099caae36c This is updated from time to time. We use a version dated 2013-07-02.
  7. Interim Register of Marine and Nonmarine Genera (IRMNG), from CSIRO. Location: http://www.cmar.csiro.au/datacentre/downloads/IRMNG_DWC.zip Updated from time to time. We use a version dated 2014-01-31.

Where taxonomies conflict regarding taxon relationships, a priority ordering is respected, the same as the order in which sources are listed above.

Any errors should be assumed to have been introduced by the Open Tree of Life project until confirmed as originating in the source taxonomy.
